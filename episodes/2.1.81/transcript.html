Claude Code v2.1.81 - March 20, 2026

Full changelog | Official docs

New Features

  • Bare mode - --bare flag for scripted -p calls skips hooks, LSP, plugin sync, skill walks. Great for CI/automation.
  • Channels permission relay - Channel servers can forward tool approval prompts to your phone for remote approval.

Notable Fixes

  • Concurrent session auth - multiple sessions no longer force repeated re-auth when one refreshes OAuth
  • Voice mode no longer swallows retry failures with misleading "check your network" message
  • Voice audio recovers when server silently drops WebSocket
  • Background agent race condition - task output hanging between polling intervals
  • Worktree session resume now switches back to the worktree automatically

Improvements

  • MCP tool calls collapse into single "Queried server" line (expand with Ctrl+O)
  • Claude suggests ! bash mode when you need interactive commands
  • Ref-tracked plugins re-clone on load for freshness
  • MCP OAuth supports CIMD for servers without Dynamic Client Registration
  • Plan mode hides "clear context" by default
  • Line-by-line streaming disabled on Windows/WSL due to rendering issues
