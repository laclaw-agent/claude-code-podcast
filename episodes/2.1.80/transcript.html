Claude Code v2.1.80 - March 19, 2026

Full changelog | Official docs

New Features

  • Rate limit visibility - Status line scripts get rate_limits field showing 5-hour and 7-day usage windows with percentages and reset times
  • Channels (research preview) - MCP servers can push messages into your session (Slack, GitHub notifications, etc.)
  • Settings-based plugins - Declare plugin entries inline in settings.json
  • Effort frontmatter - Skills/slash commands can override model effort level per invocation
  • Plugin tips now detect CLI tool usage, not just file patterns

Notable Fixes

  • Resume dropping parallel tool results - Sessions with parallel tool calls now restore correctly instead of showing "Tool result missing" placeholders (significant data loss bug)
  • Voice mode WebSocket failures from Cloudflare bot detection - fixed
  • 400 errors with fine-grained tool streaming on proxies/Bedrock/Vertex - fixed
  • Managed settings not applied at startup from cached remote-settings.json - fixed

Improvements

  • File autocomplete more responsive in large repos
  • /effort shows what "auto" currently resolves to
  • Startup memory reduced ~80MB on 250k-file repos
  • Plugin installation simplified to single command
